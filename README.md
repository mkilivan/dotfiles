# dotfiles

Personal dotfiles managed with [chezmoi](https://www.chezmoi.io/).

## Bootstrap

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply mkilivan
```

### Linux (bash) only

After `chezmoi apply` completes, add one line to `~/.bashrc` to source personal customizations:

```bash
echo '[ -f ~/.bashrc_custom ] && source ~/.bashrc_custom' >> ~/.bashrc
```

This is a one-time manual step. `~/.bashrc_custom` is generated by chezmoi from `dot_bashrc_custom.tmpl`.

> macOS users: the default shell is zsh — `~/.zshrc` is managed by chezmoi directly, no extra step needed.

> macOS prerequisite: [Homebrew](https://brew.sh/) must be installed before running `chezmoi apply` — the install script uses it for zoxide, fzf, eza, and the Nerd Font.

## Font setup

The install script automatically installs **JetBrains Mono Nerd Font**. After it runs, set this font in your terminal emulator's font settings for prompt icons to render correctly.
